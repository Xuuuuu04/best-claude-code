## 10. 本季关键拐点（[已验证]，对 Harness 团队影响大）

1. **Skills 范式三家共识**（Claude / Codex / Gemini）— 工具管理正从 function 转向 skill
2. **Sandbox Agents 成为标配**（Claude Code + OpenAI SandboxAgent）— Agent 必须有持久化 workspace
3. **1M context 下沉**（Sonnet 4.6 beta / Grok 2M / Gemini 2.5 flash 1M）— RAG 压力增大
4. **Opus 降价 3×**（$15→$5 输入，$75→$25 输出）— 重新考虑"只有重要用 Opus"的成本模型
5. **MCP 成为三家协议**（Anthropic / Google / xAI）— 集成标准定型
6. **Agent Swarm 抽象落地**（Kimi K2.5）— 多 agent 模式再前进一步
7. **Claude Code 成为 harness 反应堆**（160K+ 30K+ 30K stars 级二次创作）
