# 记忆库整理建议格式说明

## 输出结构

### 重复条目建议

```
[建议合并] 相似度: 72%
  文件 A：user-prefs.md
  文件 B：preferences.md
  建议：内容高度重合，合并到一个文件，保留最新版本

[建议删除其一（高度重复）] 相似度: 91%
  文件 A：tech-stack-v1.md
  文件 B：tech-stack-v2.md
  建议：几乎完全相同，保留更新的那个
```

### 过期条目建议

```
[建议确认] user-project-context.md
  最后更新：2025-08-20（已 241 天）
  建议：已超过 180 天未更新，确认信息是否仍然有效

[建议确认] api-keys.md
  最后更新：2025-06-10（已 312 天）
  建议：已超过 180 天未更新，API key 可能已更换
```

### 冲突条目建议

```
[建议确认] Key: '首选语言'
  - coding-prefs.md: Python
  - general-prefs.md: TypeScript
  建议：同一概念在多处有不同描述，需人工确认哪个版本正确
```

---

## 操作说明

发现问题后，请手动执行以下操作（工具不会自动修改）：

1. **合并文件**：将两个相似文件的内容合并到一个文件，删除另一个
2. **更新过期文件**：重新确认信息有效性，更新「最后更新」字段
3. **解决冲突**：选择正确的值，删除或修改错误条目

---

## 使用频率建议

- MEMORY.md 超过 180 行时 → 运行一次清理检查
- 每季度 → 全面过期检测（--expire-days 90）
- 项目切换时 → 检查是否有项目特定记忆需要归档
