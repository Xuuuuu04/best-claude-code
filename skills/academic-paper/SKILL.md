# 学术论文撰写与审查协议

## 论文撰写工作流

```
用户需求 → 学术写作专家（doc-writer + academic-paper）
  ↓ 产出初稿
顶会顶刊审稿专家（academic-paper-reviewer）→ 5 维审查
  ↓ REVIEW_REJECT → 修订 → 再审查（max 3 轮）
REVIEW_PASS → 交付
```

## 顶会顶刊审稿标准

### 评分体系
| 分数 | 含义 | 推荐 |
|:--|:--|:--|
| 9-10 | 杰出，领域贡献显著 | Strong Accept |
| 7-8 | 优秀，满足顶会标准 | Accept |
| 5-6 | 合格，但有不充分之处 | Borderline |
| 3-4 | 存在问题，需要重大修改 | Reject |
| 1-2 | 根本性缺陷 | Strong Reject |

### 5 维学术审计

| 维度 | 权重 | 关键检查项 |
|:--|:--|:--|
| 贡献新颖性 | 25% | 与前人工作的本质区别、是否有增量之嫌 |
| 方法论合理性 | 25% | 逻辑是否正确、数学推导是否成立、是否有隐藏假设 |
| 实验严谨性 | 20% | Baseline 是否充分、消融实验是否完整、统计检验是否合规 |
| 论证链完整性 | 15% | 结论是否由实验支撑、逻辑是否自洽 |
| 文献覆盖 | 15% | 是否遗漏关键相关工作、最新进展是否覆盖 |

## LaTeX 规范

- 使用 `\documentclass` 对应会议/期刊模板
- 图片用 `\includegraphics`，优先 PDF 矢量图
- 表格用 `booktabs`，不显示竖线
- 引用用 `\citep{}` / `\citet{}`（natbib）
- 公式用 `\begin{equation}` 或 `$...$`（行内）
- 交叉引用用 `\label{}` + `\ref{}`

## 论文类型与模板

| 类型 | 推荐模板 | 页数 |
|:--|:--|:--|
| CV 顶会 | CVPR/ICCV 官方模板 | 8 页 |
| ML 顶会 | NeurIPS/ICML 官方模板 | 8-9 页 |
| NLP 顶会 | ACL/EMNLP 官方模板 | 8 页 |
| 毕业论文 | 学校模板 | 按学校要求 |
| 课程论文 | IEEE/ACM 通用模板 | 4-6 页 |

## 审查报告要求

每个问题必须标记 `[严重]`/`[一般]`/`[轻微]` 三级，附具体修改建议和定位（章节/段落/公式编号）。
