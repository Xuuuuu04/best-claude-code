# PDF 工作流协议

## 工作流

1. 判断 PDF 类型：文本型、扫描型、表单、图文混排、受保护文件。
2. 读取时保留页码定位；表格、图片、脚注和页眉页脚单独说明。
3. 扫描件先做 OCR 可行性判断；低质量 OCR 必须标注置信风险。
4. 编辑/合并/拆分后验证页数、页面顺序、旋转方向、表单字段、水印和文件可打开。
5. 摘要必须区分原文事实、推断和建议。
6. 汇报输出路径、页码范围、处理方法、验证结果和无法保证的内容。

## 约束

- 不声称 PDF 原生可编辑；必要时说明格式限制。
- 不省略页码引用。
- 不绕过加密或权限限制。
